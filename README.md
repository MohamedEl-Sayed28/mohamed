### to remove dev branch locally 
- git branch -d dev
### to remove dev branch remotely 
- git push origin :dev

### to list tags  
- git tag


### to delete tag locally 
- git tag -d (tag name)
### to delete tag remotely
- git push origin --delete (tag name)
